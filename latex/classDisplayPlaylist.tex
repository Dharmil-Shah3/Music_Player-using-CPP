\doxysubsection{Detailed Description}
The Display\+Data class. 

Objective of the class is to display the details continously of the song currently being played.~\newline
It maintains a queue of type \doxylink{classSong}{Song} class named \char`\"{}playlist\char`\"{}.~\newline
Objects of class \doxylink{classSong}{Song} are pushed into that queue.~\newline
~\newline
 display\+Song\+Details() takes care of the display operation.~\newline
~\newline
 \doxylink{classDisplayPlaylist_a5ca0279c9e53c247308756030cd69e99}{play\+Next\+Song()} pops the first song from the playlis(queue) after it is played successfully.~\newline
~\newline
 check\+For\+Exception() waits until any exception is raised or occured, and prints the exception and returns value 1.
