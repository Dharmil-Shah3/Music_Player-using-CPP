\doxysubsection{Detailed Description}
The \doxylink{classSong}{Song} class represents a song, and contains song related attributes and methods. 

The \doxylink{classSong}{Song} class contains song attributes such as
\begin{DoxyEnumerate}
\item unique id of the song (auto-\/generated)
\item Name of the song
\item Duration of the songs (in chrono seconds)
\item thubnail\textquotesingle{}s path of the song
\end{DoxyEnumerate}In addition, it has also a static attribute named total\+Songs.~\newline
total\+Songs is used to provide the auto-\/generated id\textquotesingle{}s to each object of the \doxylink{classSong}{Song} class.~\newline
initially it\textquotesingle{}s zero(0), but it\textquotesingle{}s value is being incremented in the constructor of \doxylink{classSong}{Song} class.~\newline
~\newline
 It also provides the getter methods to get all attribute\textquotesingle{}s value listed above, except total\+Songs,~\newline
since all the attributes are private.
