priority variable is used to compare with priority of the log message to be displayed. Before displaying the log, this variable is used to check and compare with the priority of the log message is equal or higher than this or not.~\newline
If the priority of the log message is lesser that this variable, then that log message will not be displayed.~\newline
Default value is Log\+Priority\+::\+Debug.
