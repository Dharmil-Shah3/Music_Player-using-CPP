\doxysubsection{Detailed Description}
\doxylink{classLogger}{Logger} class is used for trace logging and debugging purpose. 

This class provides members and methods to display and save(into files), logs with different types/priorities.~\newline
This class uses values of {\ttfamily enum Log\+Priority} to display the differernt type of logs.~\newline
This class uses std\+::mutex to be a {\bfseries{thread safe}} class.~\newline
\href{https://youtu.be/1rnmYBSppYY}{\texttt{ Reference You\+Tube video}}.~\newline
~\newline
 Along with the priority and log message, this also displays
\begin{DoxyEnumerate}
\item thread-\/id
\item line number
\item function name 
\end{DoxyEnumerate}
